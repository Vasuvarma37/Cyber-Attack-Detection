\# Explainable AI-Based Cyber Attack Detection Using Machine Learning and Anomaly Analysis



\## Overview



This project develops an Explainable Artificial Intelligence (XAI) framework for cyber attack detection using Machine Learning and Anomaly Detection techniques.



The system identifies malicious network activities and provides human-readable explanations for its predictions using SHAP.



\## Features



\* Cyber Attack Detection

\* Anomaly Detection

\* Explainable AI (SHAP)

\* Random Forest Classification

\* XGBoost Classification

\* Streamlit Dashboard

\* Research-Oriented Architecture



\## Dataset



\* CICIDS2017 Dataset
